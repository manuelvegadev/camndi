# NDI SDK Files

Place the following files from the NDI SDK in this directory:

1. **Processing.NDI.Lib.h** (and any other SDK headers) — replace the stub header
2. **libndi.dylib** — the NDI runtime library

## Download

Get the NDI SDK from: https://ndi.video/for-developers/ndi-sdk/

After installing, the files are typically located at:
- macOS: `/Library/NDI SDK for Apple/`

Copy the headers and `libndi.dylib` into this `NDI/` folder.
